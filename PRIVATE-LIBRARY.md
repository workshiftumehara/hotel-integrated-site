# Private Design Library

This site is intended to be viewed locally on this PC.

## Open it

Double-click:

```text
open-private-library.cmd
```

Then open:

```text
http://127.0.0.1:3045/
```

## Why this is private

The private start command binds the server to `127.0.0.1`, which means it is reachable from this computer only.

Do not deploy this project publicly unless you want other people to see the design library.
